# Mysql

## Installation

### Windows MySQL Installer

https://dev.mysql.com/downloads/installer/

### Mac OS X by brew

#### Install

``` bash
brew install mysql
```

#### Start

``` bash
brew services start mysql
```

#### Stop

``` bash
brew services stop mysql
```
